# Career Quota Tracker

A small web app for tracking NSS career development requirements over a two-week period.

## Bi-weekly goals

- 4+ job applications
- 6+ GitHub commits
- 8+ LinkedIn networking connections

## Purpose

This project helps document job search activity, networking efforts, and GitHub progress in one clean dashboard.
## Live Project

https://duleywilliams.github.io/career-quota-tracker/

## Project Status

This project was built as part of my NSS career development tracking process. It tracks job applications, GitHub commits, and LinkedIn networking activity for each two-week reporting period.
